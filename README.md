# 🚀 Machine Gnostics Workshop at UCT

**Breaking the Ice with Machine Learning & Understanding Modern ML Approaches**

A comprehensive hands-on workshop designed to teach machine learning fundamentals through practical examples, with a focus on comparing traditional methods (Scikit-Learn) with the modern Machine Gnostics framework.

---

## 📚 What is This Workshop?

This workshop is a **three-part progressive learning experience** that takes you from ML basics to advanced techniques. You'll learn:

- How linear regression works and why degree polynomials matter
- The **Bias-Variance Tradeoff** and overfitting problems
- Why **Machine Gnostics outperforms traditional ML** on noisy data
- **Cross-validation**, convergence analysis, and residual entropy
- Best practices for real-world machine learning projects

Each notebook builds on the previous one, with hands-on code examples and interactive visualizations.

---

## 🎯 Learning Goals

By the end of this workshop, you will:

✅ **Understand core ML concepts**

- What regression is and how it works
- The role of bias and variance in model errors
- Why overfitting is a problem and how to detect it

✅ **Master two major approaches**

- Traditional: Scikit-Learn (industry standard)
- Modern: Machine Gnostics (robust, insightful)

✅ **Apply validation techniques**

- Train-test splitting
- K-fold cross-validation
- Comparing model performance fairly

✅ **Interpret advanced metrics**

- Mean Squared Error (MSE) and Mean Absolute Error (MAE)
- R² scores and residual analysis
- Residual entropy (unique to Machine Gnostics)

✅ **Build production-ready workflows**

- Handling outliers and noise
- Choosing the right model complexity
- Making informed decisions with data

✅ **Compare ML frameworks**

- Know the strengths and weaknesses of each approach
- Decide which tool is best for your problem
- Understand the mathematical foundations
